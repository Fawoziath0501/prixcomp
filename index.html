Ce que nous faisons

Des fonctionnalités pensées
pour vraiment vous aider

Chaque fonctionnalité répond à un problème concret : trouver le meilleur prix sans passer des heures, ne jamais atterrir sur un produit indisponible.

⚡
Comparaison multi-plateformes en temps réel
Sur chaque fiche produit, un tableau classe automatiquement toutes les sources par prix croissant. La mise à jour se fait toutes les 6 heures via nos jobs automatisés.
🔍
Vérification automatique des disponibilités
Un système toutes les 6h vérifie que chaque lien mène bien à un produit disponible. Si une source devient indisponible, elle est masquée automatiquement — vous ne serez jamais redirigé vers une page morte.
📊
Historique des prix
Chaque changement de prix est enregistré. Visualisez l'évolution du prix dans le temps et détectez les vraies promotions des fausses remises gonflées.
🎯
Offres flash et bons plans
Une section dédiée aux produits avec les remises les plus importantes du moment, mise à jour toutes les 30 minutes. Filtrage par catégorie et plateforme.
📖
Guides d'achat experts
Des guides rédigés pour vous aider à choisir : quel smartphone acheter pour votre budget, les meilleurs laptops pour étudiants, les pièges à éviter lors d'achats en ligne.
🔔
Alertes par email
Inscrivez-vous à la newsletter et soyez alerté en priorité lors des meilleures offres. Vous recevez uniquement ce qui correspond à vos catégories préférées.
🏷️

Chaque lien passe par notre système de tracking court (/go/xxxx) avant la redirection. Vous savez toujours où vous allez. Nous percevons une commission sur les achats — c'est ainsi que le site reste gratuit.

Multi-devises (XOF, EUR, NGN…)
Les prix sont affichés dans la devise locale de chaque plateforme. Pour les achats internationaux, une indication du coût de livraison est fournie.
📱
100% mobile optimisé
Interface entièrement responsive, pensée pour les connexions mobiles en Afrique de l'Ouest. Chargement optimisé, images adaptatives, navigation tactile fluide.

Technologie

Une architecture robuste derrière chaque prix

Le site est construit sur Laravel avec un système de jobs automatisés qui garantit des données fraîches et des liens valides à toute heure.

Import automatique des flux produits
Un scheduler déclenche les imports (API, XML, CSV) selon la fréquence configurée par plateforme. Les nouveaux produits arrivent en brouillon et sont vérifiés avant publication.
Vérification de disponibilité toutes les 6h
Chaque lien affilié est testé régulièrement par une requête HEAD légère. En cas d'échec répété, la source est masquée et une alerte est envoyée.
Historique complet des prix
Chaque changement de prix est enregistré en base avec horodatage. La table product_prices permet de tracer l'évolution complète dans le temps.
Architecture multi-providers extensible
Le pattern Strategy permet d'ajouter de nouvelles plateformes (Cdiscount, Fnac, Kilimall…) en créant un seul fichier Provider sans modifier le reste du système.
// Exemple : import automatique Jumia BJ // Déclenché toutes les heures par le scheduler AffiliatePlatform::needsSync()->get() ->each(fn($platform) => FeedImportJob::dispatch($platform) ); // Chaque produit importé est créé en brouillon Product::create([ 'is_active' => false, // invisible front 'name' => $data['name'], 'currency' => 'XOF', ]); // Vérification disponibilité — job léger // HEAD request, 3 échecs → masquage + alerte $response = Http::head($affiliateUrl); if (!$response->successful()) { $source->update(['is_active' => false]); Mail::to($admin)->send( new ProductUnavailableMail($source) ); }
